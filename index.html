Tahfizh Dashboard

Sistem pengelompokan halaqoh otomatis berbasis web.

Node.js Express MySQL MVC Architecture

Algoritma Clustering

Proyek ini mengimplementasikan K-Means Clustering untuk mengorganisir data halaqoh dengan parameter optimasi:

  • Manhattan Distance: Digunakan untuk menghitung kedekatan antar data tahfizh.
  • Silhouette Score: Digunakan untuk validasi dan penentuan jumlah cluster terbaik.

Informasi Proyek

Versi asli aplikasi ini menggunakan database tahfizh dan memerlukan konfigurasi file .env untuk berjalan secara penuh [1]. Halaman ini adalah representasi statis untuk keperluan dokumentasi di GitHub Pages.

Catatan Pengembang: Pastikan untuk mengabaikan folder node_modules dan file .env dalam produksi sebenarnya untuk menjaga keamanan data [3].
