Halaqoh Clustering System v2

Tentang Proyek

Proyek ini adalah sistem pengelolaan data Tahfizh menggunakan algoritma K-Means Clustering untuk pengelompokan halaqoh [1, 2].

Metode yang digunakan:

  • Manhattan Distance - Untuk menghitung jarak antar titik data [2].
  • Silhouette Score - Untuk mengevaluasi kualitas cluster yang terbentuk [2].

Status Pengembangan

Versi ini adalah Static Preview. Aplikasi lengkap menggunakan Node.js, Express, dan MySQL [1].

Untuk menjalankan versi penuh, silakan clone repositori ini dan ikuti instruksi di README.md [1].
